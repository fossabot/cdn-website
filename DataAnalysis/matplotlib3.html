2. pie图¶

explode：设置各部分突出
label:设置各部分标签
labeldistance:设置标签文本距圆心位置，1.1表示1.1倍半径
autopct：设置圆里面文本
shadow：设置是否有阴影
startangle：起始角度，默认从0开始逆时针转
pctdistance：设置圆内文本距圆心距离
返回值
l_text：圆内部文本，matplotlib.text.Text object
p_text：圆外部文本

label_list = ['创业团队','风险投资', '无形资产入股', '专家老师顾问团']    # 各部分标签
size = [200,100,50,50]    # 各部分大小
color = ["red", "green", "blue",'cyan']     # 各部分颜色
explode = [0,0.1,0,0]   # 各部分突出值

patches, l_text, p_text = plt.pie(size, explode=explode, colors=color, labels=label_list, 
                                  labeldistance=1.1, autopct="%1.1f%%", shadow=True, startangle=60, pctdistance=0.5)
plt.legend()

plt.axis("equal")    # 设置横轴和纵轴大小相等，这样饼才是圆的
plt.title('股权分配',fontsize=23)
plt.show()

3. 散点图¶

s：调整散点的大小，大小可以写10……,后面赋值一个变量也可以随机数大小的图形
c：如果用color的话就得用#00ffff；如果用c的话直接是b
marker：改变图标的的相貌有圆形，小三角……
注意cmap的影响，调色板这里要解释一下，大家可能注意到了：图片中的红色倒三角，面积越大的颜色越红。这是因为我们在ax.scatter()中启用了参数cmap，它需要与控制颜色的参数c配合使用。cmap指明调色板的类型，c指明颜色的深浅。 
调色板的类型可见：matplotlib官网解释
edgecolors：调整边界颜色===参数可以是none
